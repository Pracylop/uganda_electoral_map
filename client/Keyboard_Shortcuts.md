# Keyboard Shortcuts

## Broadcast Mode

Keyboard shortcuts available when using the Broadcast App (`/broadcast`).

### Navigation & Views

| Key | Action |
|-----|--------|
| `M` | Switch to Map view |
| `D` | Switch to Dashboard view |
| `C` | Switch to Comparison view |
| `I` | Switch to Issues view |
| `H` | Reset to national view (Home) |
| `B` | Go up one level (Back/drill up) |
| `/` | Open region search |

### Panels & Controls

| Key | Action |
|-----|--------|
| `E` | Open election selector |
| `L` | Toggle layers/map settings panel |
| `F` | Toggle issues filter panel (Issues view only) |
| `A` | Toggle annotation mode |
| `[` | Collapse sidebar |
| `]` | Expand sidebar |
| `P` | Toggle sidebar position (left/right) |

### Annotation Mode

| Key | Action |
|-----|--------|
| `Z` | Undo last annotation |
| `Y` | Redo annotation |

### System

| Key | Action |
|-----|--------|
| `F11` | Toggle fullscreen |
| `Escape` | Exit annotation mode, or exit broadcast mode |
