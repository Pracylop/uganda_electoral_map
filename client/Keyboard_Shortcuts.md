# Keyboard Shortcuts

## Broadcast Mode

Keyboard shortcuts available when using the Broadcast App (`/broadcast`).

### Navigation & Views

| Key | Action |
|-----|--------|
| `H` | Switch to Home (BroadcastHome) |
| `E` | Switch to Current Election view |
| `P` | Switch to Past Elections view |
| `M` | Switch to Map view |
| `D` | Switch to Dashboard view |
| `C` | Switch to Comparison view |
| `I` | Switch to Incidents/Issues view |
| `G` | Switch to Demographics view |
| `N` | Reset to national view |
| `B` | Go up one level (Back/drill up) |
| `/` | Open region search |

### Panels & Controls

| Key | Action |
|-----|--------|
| `S` | Open election selector |
| `L` | Toggle layers/map settings panel |
| `F` | Toggle issues filter panel (Issues view only) |
| `T` | Toggle interaction mode: Stats/View (Issues view only) |
| `A` | Toggle annotation mode |

### Sidebar

| Key | Action |
|-----|--------|
| `[` | Collapse sidebar |
| `]` | Expand sidebar |
| `\` | Toggle sidebar position (left/right) |

### Annotation Mode

| Key | Action |
|-----|--------|
| `Z` | Undo last annotation |
| `Y` | Redo annotation |

### System

| Key | Action |
|-----|--------|
| `F11` | Toggle fullscreen |
| `Escape` | Exit annotation mode, or exit broadcast mode |

## Quick Reference

### View Shortcuts Summary

| View | Key | Description |
|------|-----|-------------|
| Home | `H` | Main broadcast dashboard |
| Current Election | `E` | Live 2026 election results |
| Past Elections | `P` | Historical elections (1996-2021) |
| Map | `M` | Interactive choropleth map |
| Dashboard | `D` | National statistics dashboard |
| Comparison | `C` | Side-by-side election comparison |
| Incidents | `I` | Electoral incidents/issues |
| Demographics | `G` | Population & voter demographics |
